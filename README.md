## 功能模块

### 每日（首页）
- 日期头部 + 连续打卡天数
- 时间分配饼图（学习/工作/娱乐/运动/其他）
- 今日打卡（从每日模板自动生成）
- 饮食记录（早/午/晚/零食 + 卡路里统计）
- 习惯打卡（自定义习惯：喝水、运动、早起等）
- 睡眠记录（入睡/起床时间 + 自动计算时长）
- 今日支出快捷入口
- 今日记（每日自由文字日记）

### 任务
- 主线/支线/每日/限时四种任务类型
- 悬赏板（每日随机挑战，双倍奖励）
- 番茄钟（专注计时器，关联任务自动推进进度）

### 数据
- 余额统计 + 今日/本月收支
- 记一笔（支出/收入记录）
- 近7日完成趋势图 + 任务类型分布
- **交易复盘**（投资思考日记，记录市场想法）
- 交易记录（按时间筛选）

### 商店
- 积分商店（用积分兑换看剧/游戏/奶茶等奖励）
- 成就殿堂（15个成就解锁挑战）

### 设置
- 云同步（基于 jsonbin.io）
- 每日模板管理
- 习惯管理（新增/删除/开关）
- 个人等级信息

## 其他特性

- **暗黑/白天模式** — 右上角 ☀️/🌙 按钮切换，偏好自动保存
- **跨设备云同步** — 基于 jsonbin.io，数据变更后自动上传
- **移动端适配** — 底部 Tab 栏，单手操作友好

## 云同步设置

1. 访问 [jsonbin.io](https://jsonbin.io/login) → 注册/登录
2. 点击 **API Keys** → 复制 Master Key（以 `$2a$` 开头）
3. 打开本应用 → 点击右上角 **云同步** 按钮
4. 粘贴 API Key → Bin ID 留空 → 保存并连接
5. 系统自动创建云端存储，复制弹窗中的 Bin ID
6. 另一台设备填入相同 API Key + Bin ID 即可同步

## 技术栈

- 纯 HTML + CSS + JavaScript（单文件，无外部依赖）
- jsonbin.io（云同步后端）
- localStorage（本地持久化）

## 使用方式

直接在浏览器中打开 `nexus-core.html` 即可使用，无需安装任何依赖。
